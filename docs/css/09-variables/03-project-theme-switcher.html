How This Works

All colors are CSS variables defined on :root (light) and .dark (dark). Clicking the toggle adds/removes .dark class on <html>. Every var() reference instantly updates. Zero component CSS changes.

Variable Scoping

Variables cascade downward. :root is global. You can override on any element — children inherit the override. This card uses global variables, but you could scope locally too.
